## Add tab completion

1. clone git project
2. checkout version tag
3. cp completion file to ~/
4. add line to bashrc and source
5. add gif or img

## Decorate prompt

1. cp prompt.sh to ~/
2. add lines to bashrc
3. modify PS1
4. add gif or img
